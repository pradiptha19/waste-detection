# 🌊 Deep Learning - Based Floating Plastic Waste Detection System

## 📌 Overview
This project is an AI-powered system for detecting floating plastic waste in water bodies using YOLO. 
It performs real-time object detection, tracks detected waste, stores detection information, and sends email alerts 
when plastic waste is detected.

## ✨ Features
- Real-time plastic waste detection using YOLO
- Image preprocessing for improved detection
- Object tracking
- Email alert system
- Modular Python implementation

## 🛠️ Technologies Used
- Python
- YOLO
- OpenCV
- NumPy
- VS Code

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/pradiptha19/waste-detection.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python run_complete_system.py
```

## 📊 Output
The system detects floating plastic waste from images or video streams, stores detection information, and can generate alerts.
